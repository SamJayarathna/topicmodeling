The International Conference on Software Engineering (ICSE) is the premier software engineering conference, providing a forum for researchers, practitioners and educators to present and discuss the most recent innovations, trends, experiences and concerns in the field of software engineering.

Demos On-line

Submitted by admin on Fri, 05/27/2011 - 11:40

Did you miss a Demonstrations session?  Not to worry, there are videos on-line for all of the demos in the Demonstrations Track.   Just click on the title to view the demonstration page.  Those titles annotated with "(video)" contain an embedded video on the page.

User Accounts

Submitted by admin on Thu, 05/26/2011 - 17:34

If you haven’t done so already, consider creating a user account this site.   Authenticated users can:

  • Create a user profile and tell us about themselves
  • Generate a custom schedule
  • Bookmark papers of interest
  • Comment on papers and sessions
  • Participate in ICSE forums
  • Upload their favorite conference photos to our galleries

Conference Proceedings Download

Submitted by admin on Wed, 05/25/2011 - 01:19

Conference attendees may download a digital copy of the conference proceedings from http://proceedings.2011.icse-conferences.org.  This URL is only accessible in the event space.  If you have any problems downloading the proceedings, please see a student volunteer for help.
