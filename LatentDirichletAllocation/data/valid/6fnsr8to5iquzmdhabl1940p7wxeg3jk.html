Introduction

You are invited to participate in The International Conference on Intelligent Semantic Web - Services and Applications (ISWSA 2011) that will be held in Amman, Jordan, on April 18-20, 2011. The event will be held over three days, with presentations delivered by researchers from the international community, including presentations from keynote speakers and state-of-the-art lectures.

The ISWSA 2011 proceedings will be published in the ACM Digital Library (ACM ISBN of ISWSA 2011 is 978-1-4503-0474-0). All publications will be submitted for indexing with ULRICH Database.

The goal of this conference is to bring together people from the areas of Semantic Web, Social Web, Web Services, and Web Applications, and others to discuss the latest research trends in Semantic Web such as:

· Modeling and building Semantic Web agents
· Using Semantic Web Technologies to solve problems in E-commerce and E-government

The conference welcomes papers on the following (but not limited to) research topics:

§ Data Semantics and Ontologies
o Ontology modeling, reuse, extraction, and evolution
o Ontology mapping, merging, and alignment
o Searching and ranking ontologies
o Ontology evaluation
o Reasoning

§ Applications of the Semantic Web
o Applications with clear lessons learned or evaluations
o Semantic Web for large scale applications
o Semantic Web for desktops or personal information management
o Semantic Web technologies for multimedia, sensors, and situational awareness
o Semantic Web technologies for P2P, services, agents, grids and middleware
o Semantic Web technologies for software and systems engineering
o E-government
o Mobile Semantic Web
o Database, IR and AI technologies for the Semantic Web
o Search, query, integration, and analysis on the Semantic Web

§ Social Semantic Web
o Social networks and processes on the Semantic Web
o Semantic Web technologies for collaboration and cooperation
o Representing and reasoning about trust, privacy, and security

§ Web Applications
o Cloud computing techniques and approaches
o Frameworks for developing Web applications
o Security issues for Web applications
o Management techniques for large-scale Web applications
o Scalability issues and techniques
o Techniques for creating highly interactive Web applications
o Applications that illustrate interesting new features or implementation
techniques
o Performance measurements of Web applications
o Web services
o M-commerce applications, issues, and security

Location

ISWSA2011 will be organized by Isra University, Amman, Jordan in cooperation with The ACM Professional Chapter ACM Jordan ISWSA. All the activities of the conference will take place in Jordan.
