  • Using Acceptance Tests To Validate Accessibility Requirements In RIA (T). Willian Massami Watanabe, Renata Pontin de Mattos Fortes, Ana Luiza Dias. View slides.

Session 4: Designing Web Accessibility [video]

  • Methodology For Identifying And Solving Accessibility Related Issues In Web Content Management System Environments (T). Juan Miguel López, Afra Pascual, Cristina Menduiña, Toni Granollers. View slides.
  • Evaluation Of The Effectiveness Of A Tool To Support Novice Auditors (T). Christopher Bailey, Elaine Pearson. View slides.
  • MIPAW – Modele Of A Progressive Implementation Of Web Accessibility (C). Jean-Pierre Villain, Olivier Nourry. View slides.

  • ★ Best communication award. Mobile Sign Language Translation System For Deaf Community (C). Mehrez Boulares, Mohamed Jemni. View slides.
  • Web-content’s Syndication In Sign Language (C). Oussama El Ghoul, Nour Ben Yahia, Mohamed Jemni. View slides.
